# üß† LLM Finetuning: GPT-3.5-Turbo with Samantha DatasetThis project fine-tunes OpenAI's `gpt-3.5-turbo` on a how-to style conversation dataset from Hugging Face (`cognitivecomputations/samantha-data`). The model learns to emulate Samantha's helpful conversational tone in response to various user problems and DIY questions.---## üìÅ Dataset- Source: [`cognitivecomputations/samantha-data`](https://huggingface.co/datasets/cognitivecomputations/samantha-data)- File used: `howto_conversations.jsonl`- Format: JSON Lines  - Each record contains:    - `elapsed`: (float) Time duration    - `conversation`: (string) Dialog between "Theodore" and "Samantha"Example:```json{  "elapsed": 114.311,  "conversation": "Theodore: Hey Samantha, I have a problem with my car... \n\nSamantha: Sure, here's how to fix it..."}
